1.注册github，安装git，使用ssh-keygen -t rsa -C "2207577736@qq.com"生成key
2.使用ls -al ~/.ssh查看是否生成key
3.验证并访问git：ssh  -T git@github.com
4.设置用户和邮箱：git config --global user.name ""
                              git config --global user.email""
5.添加远程地址：git remote add origin git@github.com:liuzongjin/ceshi.git（在仓库目录下）
